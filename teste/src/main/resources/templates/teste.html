Teste de envio

Mande agora mesmo uma mensagem direto pro Telegram para verificar que tudo está funcionando. Pode anexar foto, PDF, áudio — qualquer arquivo até 20 MB.

Dicas
  • Escolha um chat já cadastrado ou cole um Chat ID qualquer no campo customizado.
  • Se anexar um arquivo: imagens (jpg/png/webp) vão como sendPhoto; o resto vai como sendDocument.
  • Se preferir não fazer upload, cole uma URL pública — o Telegram baixa direto. Útil pra mandar mapa, gráfico, etc.
  • O texto suporta Markdown: *negrito*, _itálico_, `código`, [link](url).
  • Esse envio é imediato — não passa pelo dedup do sistema; não vira histórico.
