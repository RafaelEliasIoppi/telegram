Comandos do Bot
Comando Descrição
/start Inicia a interação com o bot e registra o chat.
/status Exibe o status atual do bot e configuração do chat.
/historico Mostra os alertas mais recentes.
/fontes Configura quais fontes de alerta o chat recebe.
/nivel Configura o nível mínimo (INFO / AVISO / CRITICO).
/verificar Executa uma verificação manual das fontes.
