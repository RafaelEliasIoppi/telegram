Filtros de Assunto

Cada filtro é um trecho de texto que o monitor procura no assunto dos e-mails novos da sua caixa Gmail. Quando um e-mail bate com pelo menos um filtro ativo, um alerta formatado é enviado para todos os chats do Telegram cadastrados.

Como funciona
  • Você cadastra quantos filtros quiser (ex.: "Urgência Renal", "Convocação Plantão", "Resultado Exame").
  • O monitor Gmail roda a cada 30 min e lê todos os filtros ativos do banco — não precisa reiniciar a aplicação.
  • A busca ignora acentos e maiúsculas/minúsculas. Ex.: urgencia renal casa com URGÊNCIA RENAL.
  • É busca parcial: o filtro precisa estar contido no assunto, não precisa ser igual.
  • Use o botão toggle para pausar um filtro sem apagar (útil pra testar).
  • Clique Verificar Agora no Dashboard para forçar um ciclo de leitura imediato.
